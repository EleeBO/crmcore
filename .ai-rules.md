# AI Agent Rules for CodeRush2

> **CRITICAL**: This project uses `just` for automation.
> ALWAYS run `just --list` before generating shell commands.
> Use `just search <query>` for code retrieval to save tokens.

## Before ANY Task

1. **Check available commands**: `just --list`
2. **Understand project**: `just context`
3. **Search existing code**: `just search <pattern>`
4. **Check environment**: `just status`

## Command Preferences

| Instead of... | Use... |
|---------------|--------|
| `pytest` | `just test-backend` |
| `npm test` | `just test-frontend` |
| `ruff check .` | `just lint-python` |
| `grep -r "pattern"` | `just search "pattern"` |
| `tree` | `just context` |
| `git log` | `just wtf <file>` |
